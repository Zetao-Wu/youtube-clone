## Youtube Clone

Implemented a YouTube clone using Material-UI (MUI) for responsive design and efficient component styling, React and JSX for video streaming, search functionality, and real-time data integration via RapidAPI, adhering to best practices for maintainability and reusability.
You can click on this link to visit the deployed site: https://zetao-wu.github.io/youtube-clone/


## Project Installation

Please follow these steps to step up the webpage locally on your device.

**Make sure you have the following installed on your device before attempting to set up the project:**

- Git
- Node.js
- Node Package Manager
